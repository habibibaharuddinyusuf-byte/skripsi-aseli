### ğŸ§¹ 2. **Mengumpulkan dan Menyiapkan Dataset**

CNN membutuhkan banyak data gambar untuk belajar pola.

Langkah-langkahnya:

* **Kumpulkan data** (misal dari Kaggle, Google Dataset, atau hasil sendiri)
* **Labeli data** sesuai kelas (misal: folder `cats/` dan `dogs/`)
* **Pisahkan dataset** menjadi:

  * Training set (misal 70%)
  * Validation set (misal 15%)
  * Test set (misal 15%)
* **Preprocessing data:**

  * Resize gambar ke ukuran seragam (misal 128Ã—128)
  * Normalisasi pixel (0â€“1)
  * Data augmentation (rotasi, flip, dsb.) untuk menambah variasi

---

### ğŸ§  3. **Membangun Arsitektur CNN**

CNN biasanya terdiri dari beberapa layer utama:

* **Convolution Layer:** mengekstraksi fitur dari gambar (tepi, pola, bentuk)
* **Pooling Layer:** mengecilkan ukuran fitur map (misal: MaxPooling)
* **Flatten Layer:** mengubah hasil konvolusi menjadi vektor
* **Fully Connected (Dense) Layer:** melakukan klasifikasi berdasarkan fitur
* **Output Layer:** jumlah neuron = jumlah kelas

Contoh sederhana (dengan Keras/PyTorch):

```python
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Conv2D, MaxPooling2D, Flatten, Dense

model = Sequential([
    Conv2D(32, (3,3), activation='relu', input_shape=(128,128,3)),
    MaxPooling2D(2,2),
    Conv2D(64, (3,3), activation='relu'),
    MaxPooling2D(2,2),
    Flatten(),
    Dense(128, activation='relu'),
    Dense(2, activation='softmax')  # 2 kelas: kucing vs anjing
])
```
